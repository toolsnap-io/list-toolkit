# List Toolkit

A free online toolkit for list manipulation — merge, deduplicate, sort, shuffle, and transform text lists with ease.

## Features

- **Merge Lists** — Combine multiple lists into one
- **Remove Duplicates** — Deduplicate any text list
- **Sort** — Alphabetical, numerical, or custom order
- **Shuffle** — Randomize list order
- **Filter** — Remove empty lines or match patterns
- **Prefix/Suffix** — Add text before or after each line
- **Number Lines** — Add line numbers

## How to Use

1. Paste your list into the input area
2. Choose the operation you want to perform
3. Configure any options
4. Get instant results — copy or download
